# Employee Management System (COMP3133 Assignment 2)

## Features
- User Signup & Login
- Employee CRUD (Create, Read, Update, Delete)
- Search by department/designation
- Image upload using Cloudinary
- Authentication using token

## Technologies
- Angular
- Node.js / Express
- GraphQL
- MongoDB
- Cloudinary

## How to Run

### Backend
cd comp3133_assignment1  
npm install  
npm run dev

### Frontend
cd 101419420_comp3133_assignment2  
npm install  
ng serve  

## Deployment

Due to time constraints, the application was tested locally.
Backend and frontend are fully functional when run locally.
